About This Project

Project Overview

This data visualization platform was developed as part of the advanced web development course. It aims to provide an intuitive interface for users to upload, analyze, and visualize various datasets through interactive charts and graphs. The project demonstrates the application of modern web technologies including HTML5, CSS3, JavaScript, and Chart.js.

Key Features

  • Interactive data visualization with multiple chart types
  • CSV file upload and parsing capabilities
  • Responsive design that works across all devices
  • Modern UI with glassmorphism effects
  • Dynamic chart generation based on dataset characteristics

Technologies Used

HTML5 CSS3 JavaScript Chart.js Flask
